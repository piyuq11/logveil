# Roadmap

The project stays deliberately small and local-first.

## Near term

- Collect false-positive and missed-secret reports using synthetic examples.
- Add opt-in detectors only when their formats are documented and testable.
- Publish signed release artifacts and a reproducible installation example.

## Later

- Support configurable detector profiles without changing the default rules.
- Add a machine-readable report format for CI pipelines.
- Evaluate integrations for common log collectors while keeping the core dependency-free.

Suggestions should explain the problem, include a safe synthetic input, and
describe the expected output.
